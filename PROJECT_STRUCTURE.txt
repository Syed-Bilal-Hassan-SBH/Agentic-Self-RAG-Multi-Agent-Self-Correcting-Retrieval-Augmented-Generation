📊 WHAT STREAMING EVALUATION DOES

INPUT: 500 HotpotQA samples

PROCESSING (30-45 minutes):
├─ Process each sample through 5 RAG systems
├─ Compute metrics (EM, F1, time)
├─ Append to CSV and log files in real-time
├─ Show real-time progress bar per batch
└─ Generate final summary when done

OUTPUT FILES:
├── sidebyside_results_500.csv
│   └─ 500 rows (one per sample) x 20 columns
│      All systems visible in one row
│      Perfect for Excel comparison
│
├── detailed_results_500.csv
│   └─ 2,500 rows (5 per sample) x 9 columns
│      Traditional format for analysis
│
├── streaming_log_500.txt
│   └─ Real-time detailed sample logs
│      Human-readable with ✓/✗ indicators
│
└── streaming_summary_500.json
    └─ Final aggregated statistics

💻 EXAMPLE USAGE

RUN STREAMING EVALUATION:
────────────────────────
$ python run_streaming_evaluation.py

MONITOR PROGRESS (separate terminal):
──────────────────────────────────────
$ Get-ChildItem results/batch_evaluation/*.csv | Format-Table Name, Length

ANALYZE IN PYTHON:
──────────────────
import pandas as pd
df = pd.read_csv('results/batch_evaluation/sidebyside_results_500.csv')

# Compare all systems
for system in ['Vanilla RAG', 'Agentic Self-RAG', 'Ultra Agentic RAG']:
    em = df[f'{system}_em'].mean() * 100
    print(f"{system}: {em:.1f}%")

# Find Agentic wins
wins = df[(df['Agentic Self-RAG_em'] == 1) & (df['Vanilla RAG_em'] == 0)]
print(f"Agentic beats Vanilla: {len(wins)} times")
