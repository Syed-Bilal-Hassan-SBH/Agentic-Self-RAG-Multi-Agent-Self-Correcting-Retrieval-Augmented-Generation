EVALUATION METRICS DEFINITIONS
================================

Exact Match (EM): Binary metric (0/1)
  - 1.0 if predicted answer exactly matches gold answer
  - 0.0 otherwise
  - Range: [0, 1]
  - Interpretation: Stricter measure of correctness

F1 Score: Token-level overlap metric
  - Harmonic mean of precision and recall on token level
  - Range: [0, 1]
  - Interpretation: More lenient measure, accounts for partial correctness

Inference Time: Latency per sample (seconds)
  - Measured end-to-end inference time
  - Includes retrieval + generation
  - Range: [0.5, 10.0] seconds

Context Overlap: Percentage of relevant context retrieved
  - Range: [0, 1]
  - 1.0 = all relevant context retrieved
  - 0.0 = no relevant context

Faithfulness: How well answer aligns with retrieved context
  - Range: [0, 1]
  - 1.0 = answer supported by context
  - 0.0 = answer contradicts context

Entity Hallucination Rate: Percentage of hallucinated entities
  - Range: [0, 1]
  - 0.0 = no hallucinations
  - 1.0 = all entities hallucinated

Refusal Rate: Percentage of questions refused
  - Range: [0, 1]
  - 0.0 = answer all questions
  - 1.0 = refuse all questions
